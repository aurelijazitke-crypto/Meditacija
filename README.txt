Aurelija meditacijos landing page

Failai:
- index.html -> pagrindinis landing page
- privacy.html -> privatumo politika
- assets/ -> klientų atsiliepimų nuotraukos

Kaip paleisti:
1. Įkelk visą aplanką į GitHub repozitoriją.
2. Pajunk repozitoriją prie Vercel.
3. Vercel automatiškai paleis kaip statinį puslapį.

Svarbu prieš publikavimą:
1. privacy.html faile pakeisk [įrašyk savo el. pašto adresą] į savo tikrą el. paštą.
2. index.html formoje šiuo metu yra demonstracinis pateikimas be realaus el. pašto surinkimo.
   Jei nori, kad forma rinktų el. paštus, prijunk MailerLite arba kitą formų sistemą.
3. Jei norėsi, galiu vėliau padaryti ir integruotą MailerLite versiją.
